# Daniel Rein M. Cosare — Computer Engineering & Robotics Portfolio

Welcome to my personal portfolio repository! This website is designed as an interactive, publication-style tech magazine showcasing my projects and professional experience in **Robotics Engineering**, **Embedded Systems**, **PCB Design**, and **Full-Stack Development**.

Live Development URL: [http://localhost:3000](http://localhost:3000)  
Repository: [https://github.com/Stobeeney/Daniel_Portfolio](https://github.com/Stobeeney/Daniel_Portfolio)

---

## 🛠️ Tech Stack & Architecture

- **Backend Logic**: PHP (Modular array structures, dynamic rendering)
- **Frontend Presentation**: Semantic HTML5 & Vanilla CSS (Custom grid systems, responsive flexboxes)
- **Interactive Scripting**: JavaScript ES6 (Terminal guestbook, live online simulation counters, custom cursor crosshairs)
- **Assets & Prototyping Tools**: EasyEDA, KiCad, Solid Edge, AutoCAD, Orca Slicer, Fritzing, Tinkercad

---

## 🚀 Key Features

1. **Dynamic Project Mission Logs**: Renders 11 engineering projects (including ASL translation portals, Crab Detection edge computing, custom Sumobots, and PID controllers) using optimized PHP arrays.
2. **Chronological Career Timeline**: A visual level-up timeline detailing my journey from the Philippine Robotics Team to engineering leadership and 3D printing freelancing roles.
3. **Interactive Floating Guestbook**: A custom terminal-style widget at the bottom right corner that allows users to leave messages (saved via browser `localStorage`) and shows a live active visitor count.
4. **Interactive Profile Card**: Custom Computer Engineering player stats detailing current experience levels (Level 22), project totals, and skillsets.
5. **No Emojis (Text Badges)**: Stripped of generic browser emojis and replaced with custom text-based tag badges (e.g., `[BOT]`, `[IoT]`, `[GH]`).

## 💻 Local Installation & Deployment

To run this project locally, ensure you have **PHP** installed on your system.

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Stobeeney/Daniel_Portfolio.git
   cd Daniel_Portfolio
   ```

2. **Start the PHP development server**:
   ```bash
   php -S localhost:3000
   ```

3. **Open the browser**:
   Navigate to [http://localhost:3000](http://localhost:3000) to view the portfolio.
